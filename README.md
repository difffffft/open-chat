# open-chat

open-chat是一款开源的AI对话框架，你可以用它聊天，解决问题。

专业类方向，翻译等任务，你可以很方便的定制你的GPTS。


### 系统需求

```
Python >= 3.8
MySql >= 8.0+
```

### 安装依赖

```sh
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 生成依赖
pip freeze > requirements.txt

# 运行程序
python main.py
```
